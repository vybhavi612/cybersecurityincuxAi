

**Student Attendance Management System **

### Project Overview

This is a Student Attendance Management System developed using Streamlit, Python, and SQLite. The system helps manage student registration, attendance tracking, assignment submissions, project submissions, progress tracking, and report generation through a single dashboard.

### Frontend

The frontend is developed using **Streamlit**.

Features:

* Interactive dashboard
* Student registration form
* Student login page
* Attendance management
* Assignment and project submission pages
* Student profile and photo upload
* Reports and analytics dashboard

Streamlit provides the user interface and handles all user interactions.

### Backend

The backend is developed using **Python**.

Python is used for:

* Processing user inputs
* Attendance calculations
* Student authentication
* Progress tracking
* Report generation
* File upload handling
* Database operations

### Database

The database used is **SQLite**.

Tables used:

1. Students Table

   * Student ID
   * Name
   * USN
   * Course
   * Semester
   * Email
   * Phone Number
   * Password

2. Attendance Table

   * Attendance ID
   * Student ID
   * Login Time
   * Date
   * Status (Present/Absent)

3. Progress Table

   * Progress ID
   * Student ID
   * Assignments Submitted
   * Projects Uploaded
   * Login Count

4. Submission Table

   * Submission ID
   * Student ID
   * Assignment Name
   * Project Name
   * Submission Date

### Key Features

* Student Registration
* Student Login
* Attendance Marking (Present/Absent)
* Attendance Percentage Calculation
* Student Search, Edit, and Delete
* Assignment Submission
* Project Submission
* Student Photo Upload
* Progress Tracking
* Dashboard Analytics
